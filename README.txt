== Description ==

👉 [Plugin Demo](https://www.radiustheme.com/demo/plugins/business-reviews/) | [Documentation](https://www.radiustheme.com/docs/business-reviews/business-reviews/) | [Get Pro](https://www.radiustheme.com/downloads/business-review/)  👈

By using Business Reviews plugin you can display your Google Business Reviews, Facebook reviews & recommendations and Yelp review. 

== 🏆 Google Business Reviews ==

Business Reviews plugin display your Google Business reviews and rating ie. Google Places reviews and rating on your WordPress website in list or grid view within page post or widget. This plugin save your reviews in WordPress database so no dependency of any services to display Google review on your website. So no loading time related issue as it get review from WordPress database. You can generate unlimited shortcode and use any page/ post or widget.  

You need Google MAP API key and place ID to display Google reviews. [See how to get the API key](https://www.radiustheme.com/docs/business-reviews/configurations/google-settings/how-to-get-google-places-api-key/).

== 🏆 Facebook Reviews & Recommendations ==

This plugin display Facebook reviews and recommendations on your WordPress website in list and grid view on page/ post or widget. You can generated unlimited shortcode and can use the shortcode in page/ post or call as widget. This plugin save the Facebook review in WordPress database so no dependency to display the Facebook reviews & recommendations.

It required Facebook API key to display the review see how to [create the Facebook API key](https://www.radiustheme.com/docs/business-reviews/configurations/facebook-settings/how-to-get-facebook-page-access-token-and-id/).

== 🏆 Yelp Reviews ==

This plugin display your Yelp Business reviews on your WordPress site in list/ grid view on page/ post or widget. You can generated unlimited shortcodee and can use the shortcode in page/ post or widget. This plugin save the Yelp reviews in WordPress database so there is no dependency to display reviews in your site.

It  required Yelp API key to get the Yelp reviews. See how to [create the Yelp API key](https://www.radiustheme.com/docs/business-reviews/configurations/yelp-settings/how-to-create-yelp-api-key/).

== Installation ==

1. Unzip the downloaded `business-reviews-wp` zip file
2. Upload the 'business-reviews-wp' folder and its contents into the `wp-content/plugins/` directory of your WordPress installation
3. Activate `Business Reviews` from Plugins page

= Requirements =
* **WordPress version:** >= 4.5
* **PHP version:** >= 7.4

== Frequently Asked Questions ==

= How to Use Google Facebook Yelp Review=
 
* Go to `Reviews Reviews > Settings` Set necessary API key
* Go to `Business Reviews > New Shortcode` and generator shortcode by your requirements
* Go to page or post editor insert shortcode/widget.
